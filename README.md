# 📐 Cálculo de Área, Perímetro e Diagonal de um Retângulo em Java

## 📌 Descrição

Este projeto consiste em uma aplicação em Java que realiza cálculos geométricos a partir das dimensões de um retângulo.

O usuário informa a largura e a altura, e o sistema calcula automaticamente a área, o perímetro e a diagonal do retângulo.

---

## 🚀 Funcionalidades

* Entrada de dados:

  * Largura do retângulo
  * Altura do retângulo
* Cálculo da área
* Cálculo do perímetro
* Cálculo da diagonal (utilizando fórmula matemática)
* Exibição formatada dos resultados

---

## 🧱 Estrutura do Projeto

O projeto é dividido em duas classes principais:

* `Retangulo` → Classe principal responsável pela execução do programa e interação com o usuário
* `CalculoArea` → Classe responsável pelos cálculos geométricos do retângulo
